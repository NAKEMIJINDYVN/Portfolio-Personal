# CV Submission Form
 **CV Submission Form**, a clean and user-friendly project designed to **optimize the experience of submitting your CV online**. This project demonstrates a modern, intuitive interface that allows recruiters to receive applications efficiently while maintaining a professional look.

## Highlights
- **User-Friendly:** Clear and simple input fields for **Full Name, Email**, and **CV upload**.
- **Responsive Design:** Works seamlessly on devices of all sizes, from desktops to smartphones.
- **Modern UI:** Uses the **DM Sans** font, cohesive color scheme, and interactive button hover effects for a professional feel.
- **Instant Feedback:** Displays a confirmation message immediately after submission to improve user experience.
- **Tech Stack:** HTML, CSS, JavaScript – lightweight but effective.
